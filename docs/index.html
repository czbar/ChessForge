Curated Workbook Collection

This page contains a curated collection of Chess Forge workbooks.

Some of them have been created specifically for Chess Forge while others have been adapted from books or lichess studies . All are included here with authors' permissions.

Simply download a workbook and open in Chess Forge. Note that Chess Forge workbooks are PGN files and can be opened in any PGN file viewer. However, those viewers will not take advantage of Chess Forge's PGN extensions and features.

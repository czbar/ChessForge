Workbooks

The program operates on data stored in its Workbooks. Workbooks are text files in the PGN format with some Chess Forge specific extensions. Those extensions, include extra tags identifying the file as a ChessForge workbook, extra headers identifying PGN "game" types as Studies, Games or Exercises in the Chess Forge's context etc.

Workbook files can be opened by any PGN viewer/editor but some content will not be shown.

Each workbook contains one or more chpater and within chapters:

  • Optional Intro
  • A Study
  • Any number of Games
  • Any number of Exercises

Self-indexing Study View

If you do not want indexing, set the default indexing depth to 0. Any new Study will then open without the index.

However, you can immediately enable indexing by simply clicking the Study's title.
